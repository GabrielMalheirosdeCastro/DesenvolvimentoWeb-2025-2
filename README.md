# 🌐 Interface Gráfica Pessoal - Gabriel Malheiros de Castro

> **Projeto acadêmico de portfólio pessoal desenvolvido para a disciplina de Desenvolvimento Web na FAESA (2025-2)**

Site pessoal profissional que demonstra competências em desenvolvimento web moderno, incluindo HTML5 semântico, CSS3 avançado, design responsivo e arquitetura de componentes React. O projeto converte assets do Figma para código funcional, implementando um sistema de galeria interativa, dados pessoais dinâmicos e desafio de código morse.

## 🚀 **Como Executar Localmente**

### 🔧 **Servidor de Desenvolvimento**
```powershell
# 1. Clonar o repositório
git clone https://github.com/GabrielMalheirosdeCastro/DesenvolvimentoWeb-2025-2.git

# 2. Navegar para o diretório
```

## 📚 **Detalhes dos Laboratórios Acadêmicos**

### � **Laboratório 1 - Fundamentos CSS**
Demonstração dos conceitos básicos e fundamentais do CSS:
- **🎯 Foco**: Seletores, propriedades básicas, layout
- **🔧 Conceitos**: Seletores CSS, Box Model, Display, Position
- **📐 Layout**: Flexbox e Grid básicos
- **🎨 Estilização**: Cores, bordas, espaçamentos

### �🔤 **Laboratório 2 - Tipografia e Cores**
Demonstração completa das propriedades tipográficas CSS e sistemas de cores:
- **🎯 Foco**: Propriedades tipográficas CSS avançadas
- **🔧 Propriedades**: font-family, font-size, font-weight, line-height, text-align, text-decoration, text-transform, letter-spacing
- **🎨 Sistemas de Cores**: Nomes, Hexadecimal, RGB, RGBA (com transparência), HSL
- **📱 Google Fonts**: Integração com Roboto e Poppins
- **📐 Hierarquia Visual**: Demonstração de estrutura de títulos e texto
- **♿ Acessibilidade**: Exemplos de contraste adequado e legibilidade

### � **Laboratório 3 - Box Model e Espaçamentos**
Compreensão prática do modelo de caixas CSS e propriedades de espaçamento:
- **🎯 Foco**: Box Model, margin, padding e border
- **🔧 Conceitos**: Content, padding, border, margin, box-sizing
- **📐 Demonstrações**: Visualização do Box Model, sintaxe de margin/padding
- **🎨 Exemplos Práticos**: Card de produto, margin auto, border styles
- **🏆 Desafio**: Criação de card com hover effects e box-shadow
- **♿ Acessibilidade**: DevTools para inspeção do Box Model

## 🔧 **Resolução de Problemas**

### 🚨 **Problemas Comuns e Soluções**

#### **❌ Erro: "localhost:3000 não carrega"**
```powershell
# Solução 1: Verificar se o servidor está rodando
npm run dev

# Solução 2: Se a porta estiver ocupada
npm run dev -- --port 3001
```

#### **❌ Erro: "Páginas HTML não carregam no desenvolvimento"**
```powershell
# Solução: Usar o preview para testar páginas HTML
npm run build
npm run preview
# Acessar: http://localhost:4173/galeria-naves-espaciais.html
```

#### **❌ Erro: "Build falha ou warnings CSS"**
```powershell
# Solução: Limpar cache e reinstalar
rm -rf node_modules package-lock.json
npm install
npm run build
```

#### **❌ Erro: "Site não funciona no Vercel"**
- ✅ **Verificar**: URLs do Vercel sempre funcionam (deploy automático)
- ✅ **Alternativa**: Usar o GitHub Pages como backup
- ✅ **Debug**: Verificar logs do Vercel no dashboard
