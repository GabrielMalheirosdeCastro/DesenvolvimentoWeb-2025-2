Exemplos de Seletores CSS

  • Seletor de elemento: p { ... } - afeta todos os parágrafos
  • Seletor de classe: .destaque { ... } - afeta elementos com class="destaque"
  • Seletor de ID: #titulo-principal { ... } - afeta o elemento com id="titulo-principal"
  • Seletor universal: * { ... } - afeta todos os elementos
