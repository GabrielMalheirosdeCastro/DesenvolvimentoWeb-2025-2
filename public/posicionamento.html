📍 Posicionamento Relative

position: relative - Move o elemento em relação à sua posição original no fluxo do documento.

Posição Normal
left: 50px
top: 20px Eu me movi relativamente à minha posição original
Posição Normal

💻 Código CSS:

.box-relative {
    position: relative;
    left: 50px;
    top: 20px;
    /* O espaço original é preservado */
}

🎯 Posicionamento Absolute

position: absolute - Remove o elemento do fluxo e posiciona em relação ao ancestral posicionado mais próximo.

Container com position: relative
top: 10px
left: 10px Topo Esquerda
top: 10px
right: 10px Topo Direita
bottom: 10px
left: 10px Baixo Esquerda
bottom: 10px
right: 10px Baixo Direita
Centralizado
transform Centro

💻 Código CSS:

.container-absolute {
    position: relative; /* Contexto de posicionamento */
    height: 300px;
}

.box-absolute {
    position: absolute;
}

.topo-esquerda { top: 10px; left: 10px; }
.centro {
    top: 50%; left: 50%;
    transform: translate(-50%, -50%);
}

📌 Posicionamento Fixed

position: fixed - Posiciona o elemento em relação à janela do navegador (viewport).

🔝 O header no topo desta página usa position: fixed

🎈 O botão flutuante no canto inferior direito também usa position: fixed

Header Fixed

Conteúdo que rola...

O header permanece fixo no topo

Btn

💻 Código CSS:

.header-fixo {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
    z-index: 1000;
}

.btn-flutuante {
    position: fixed;
    bottom: 30px;
    right: 30px;
    z-index: 999;
}

🔄 Posicionamento Sticky

position: sticky - Híbrido entre relative e fixed. Gruda no topo quando atinge uma posição específica.

Conteúdo antes do elemento sticky...

🔄 Eu sou STICKY! Role para baixo e veja que grudo no topo

Mais conteúdo...

Continue rolando...

O elemento sticky volta ao fluxo normal quando o container termina

💻 Código CSS:

.elemento-sticky {
    position: sticky;
    top: 100px; /* Distância do topo quando "gruda" */
    background: #3b82f6;
    z-index: 10;
}

📚 Z-index (Empilhamento)

z-index - Controla a ordem de empilhamento dos elementos posicionados.

📚 Elementos com z-index maior ficam na frente

🎛️ Clique nas caixas para alterar o z-index:

Z-index: 1 Camada 1
Z-index: 2 Camada 2
Z-index: 3 Camada 3

💻 Código CSS:

.box-z {
    position: absolute;
    z-index: 1; /* ou 2, 3, etc */
}

/* Elementos com z-index maior ficam na frente */
.z1 { z-index: 1; }
.z2 { z-index: 2; }
.z3 { z-index: 3; }

🔲 Modal Centralizado

✨ Como funciona:

  • position: fixed - Remove do fluxo, fixa na viewport
  • top: 0; left: 0; width: 100%; height: 100% - Ocupa toda a tela
  • display: flex; align-items: center; justify-content: center - Centraliza o conteúdo
  • z-index: 9999 - Fica acima de tudo

📚 Muito usado para alertas, formulários, galeria de imagens, confirmações, etc.
